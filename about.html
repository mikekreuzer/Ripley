About

Ripley ranks reddit's programming language subreddits by number of subscribers.

Ripley is an index of general purpose programming languages. Statistics packages like MATLAB, and SAS aren't included, nor are the various flavours of Shell, or PowerShell, or SQL, or HTML & CSS. I'm always happy to hear arguments to the contrary. If a language isn't included & you think it should be please let me know, either on GitHub or Twitter.

This is not an official reddit product, obviously.

The scraper code used is available on Github.

Other indexes

A few of the other language indexes and the methodologies used to compile them are:

  • PYPL, the Popularity of Programming Language Index: ranks searches for language tutorials on Google
  • The RedMonk Programming Languages Rankings: ranks questions on Stack Overflow against lines of code on GitHub
  • The TIOBE Index: uses 11 secret herbs & spices (they don't say what they use)
